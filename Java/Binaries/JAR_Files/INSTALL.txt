This directory contains jar files needed for Myro/Java.  Installation consists of
copying these files to the appropriate place.  Follow the instructions for your
platform.

Windows:
Copy RXTXScribblerComm.jar to (javahome)\jre\lib\ext
If you're using BlueJ, copy the remaining jar files and voices.txt to C:\BlueJ\lib\userlib
If you're not using BlueJ, copy the remaining jar files and voices.txt to (javahome)\jre\lib\ext
 
MacOSX:
To install Myro/Java for all users, copy all jar files and voices.txt to /Library/Java/Extensions
To install Myro/Java for one user, copy all jar files and voices.txt to ~/Library/Java/Extensions

Linux:
Copy RXTXScribblerComm.jar tp (javahome)/jre/lib/ext
If you're using BlueJ, copy the remaining jar files and voices.txt to /usr/local/bluej/lib/userlib
If you're not using BlueJ, copy the remaining jar files and voices.txt to (javahome)/jre/lib/ext


NOTE:  Don't forget to also install the platform-specific driver files, contained in
other folders.
