Parani SD100 Serial to Bluetooth Adapter Setup

One time setup / TA Instructions

  1. Use the dip switches on the outside of the device to set the baud rate to 38.4K and disable hardware flow control.
    1. The positions of the switches should be On, Off, On, Off.
  2. Power the device and connect it to the PC's serial port.
  3. Install and run the PromiWin software that ships with the device.
  4. Disable AT commands
  5. Turn on Authentication and give it a passkey. "1234" is a standard Bluetooth device default password. (There is a bug when there is no passkey).
  6. Optionally change the name of the device to something friendlier / unique

Re-pairing / Student Instructions

  1. Turn on scribbler with adapter plugged in.
    1. If scribbler LED flashes really fast, repeat TA instructions above.
    2. The Parani Standby LED should be constant amber.
  2. Press and hold Pairing button with tip of mechanical pencil until the Standby LED turns off.
    1. Now the Parani Connect LED should be blinking 3 times every 2 seconds. (This means device is able to be paired with a computer).
  3. Do the standard windows pairing.
    1. Make sure a com port number is assigned. Go to the "Properties" window for that device, then under the services tab, make sure the "generic serial" profile is checked.
  4. Now connect to robot with MSRS, Myro, or HyperTerminal.
    1. IMPORTANT: Before connecting to the robot with MSRS, always make sure that the robot is freshly restarted.
    2. NOTE: Pairing is not completed on the robot side until this connection is made.
    3. When connected, the Connect LED will be constant green.

Notes: When you disconnect from the robot, the Connect LED will flash 2 times every 2 seconds. This indicates that this device has found a pair and will only connect to that computer. It is not even discoverable by other computers. If you ever need to re-pair, just press and hold the Pairing button again until the Connect LED blinks 3 times every 2 seconds. Then repeat the re-pairing instructions above.
