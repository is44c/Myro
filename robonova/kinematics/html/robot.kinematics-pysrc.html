 22      For an n-axis manipulator C{q} is an n element vector or an m x n matrix of

 26      C{fkine} returns a 4x4 homogeneous transformation for the tool of 
 27      the manipulator.

 30      joint coordinates for a sequence of points along a trajectory.  q[i,j] is 
 31      the j'th joint parameter for the i'th trajectory point.  In this case 
 32      C{fkine} returns a list of matrices for each point 
 33      along the path. 
 34   
 35      The robot's base or tool transform, if present, are incorporated into the 
 36      result. 
 37       
 38      @type robot: Robot instance 
 39      @param robot: The robot 
 40      @type q: vector 
 41      @param q: joint coordinate 
 42      @see: L{Link}, L{Robot}, L{ikine} 
 43      """ 
 44   
 45      q = mat(q) 
 46      n = robot.n 
 47      if numrows(q)==1 and numcols(q)==n: 
 48          t = robot.base 
 49          for i in range(0,n): 
 50              t = t * robot.links[i].tr(q[0,i])

 76      Uniqueness 
 77      ========== 
 78      Note that the inverse kinematic solution is generally not unique, and  
 79      depends on the initial guess C{q} (which defaults to 0). 
 80   
 81      Iterative solution 
 82      ================== 
 83      Solution is computed iteratively using the pseudo-inverse of the 
 84      manipulator Jacobian. 
 85   
 86      Such a solution is completely general, though much less efficient  
 87      than specific inverse kinematic solutions derived symbolically. 
 88   
 89      This approach allows a solution to obtained at a singularity, but  
 90      the joint angles within the null space are arbitrarily assigned. 
 91   
 92      Operation on a trajectory 
 93      ========================= 
 94      If C{tr} is a list of transforms (a trajectory) then the solution is calculated 
 95      for each transform in turn.  The return values is a matrix with one row for each 
 96      input transform.  The initial estimate for the iterative solution at  
 97      each time step is taken as the solution from the previous time step. 
 98

105      Cartesian DOF (in the wrist coordinate frame) that will be ignored

109      of non-zero elements should equal the number of manipulator DOF.
