\subsection{Detailed Description}
This class implements all of the needed methods and components for controlling a scribbler robot. 

A port of the existing IPRE \doxyref{Scribbler}{p.}{classScribbler} Python code to C++. The original authors of the python based code are Keith O'Hara and Doug Blank.

\begin{Desc}
\item[Author:]Richard Edwards \end{Desc}
