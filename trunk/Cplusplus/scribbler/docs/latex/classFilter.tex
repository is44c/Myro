\subsection{Detailed Description}
The class defintes a frame work creating data filters which can manipulate images, and apply any changes in the image data back to the image. 

The class is used within the \doxyref{VideoStream}{p.}{classVideoStream} package to allow for the creation of various image filters, which can be applied to a stream and the results will be visible in real time.

Richard Edwards
