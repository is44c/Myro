To install these on a MacOSX 10.6 (Snow Leopard) system for all users:

Copy the following files to /Library/Java/Extensions
     RXTXScribblerComm.jar
     jinput.jar
     jutils.jar
     Myro.jar
     rxtxScribblerSerial.jnilib
     libjinput-osx.jnilib

To install these on a MacOSX 10.6 (Snow Leopard) system for one user:

Copy the following files to ~/Library/Java/Extensions
     RXTXScribblerComm.jar
     jinput.jar
     jutils.jar
     Myro.jar
     rxtxScribblerSerial.jnilib
     libjinput-osx.jnilib
