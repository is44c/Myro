Mac OSX Instructions

Installation on MacOSX is supported on the wiki.roboteducation.org website at the Mac OSX Setup Page. Please visit that page for the latest instructions. We have placed commonly used files/packages in the \MacOSX directory in case you are on a slow Internet connection, but we recommend that you download the latest files from the Internet if you have a high speed connection.

Then you should look at the Myro Reference Manual.
