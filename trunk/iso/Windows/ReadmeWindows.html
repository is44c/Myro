Windows XP and Vista Install Instructions

Installation on windows XP and Vista currently requires that you run the install.bat file as the computer's administrator, and not as a regular user.

If you are running as the default user in Windows XP, it is likely that you already have administrator privileges, so go ahead and start the install.bat file by double clicking on it.

To run the batch file in Windows Vista, do one of the following:

  1. Navigate to the CD-Rom. (Right click on the Start button and select Explore. Select Computer.)
  2. Go into the Windows folder on the CD. (Right-click on the CD icon, and select Open or Explore from the context menu.)
  3. In Windows Vista, Right-Click on the install.bat file. (A context menu will open up.) Select "Run as Administrator" from the context menu. In Windows XP you can simply double-click on the install.bat file.
or
  1. Click the Start button.
  2. In the start menu, click the All Programs icon.
  3. In the Programs menu, click the Accessories folder icon.
  4. Right-click on the Command Prompt icon (which will appear under Accessories).
  5. Then select "Run as Administrator" from the menu. A window will open up. It is referred to as the Command Prompt.
  6. In the Command Prompt window, execute the install.bat file by typing "d:\WinVista\install.bat" (without the quotes) and pressing enter. If your CD-Rom/DVD drive is a different drive letter than D, you should substitute the correct drive letter.
By running the install.bat program, you will begin installing a series of programs necessary for Myro to run. Keep clicking Next and Install until it is finished. Some general guidance:
  1. Always choose Python2.4 if given a choice
  2. If you already have Python2.4 installed, choose the Repair option
  3. Always take the recommended defaults (except were noted here)
  4. Never select Remove
  5. You must have administrator rights to install the software (and upgrade it)
  6. If you receive error messages about a missing MSVCR71.DLL file, you should install Microsoft .NET 1.1 framework. (install file included on CD)

    7. Upgrading

    After you install the software, you must upgrade to the latest version via the Internet. Double-click the "Start Python.pyw" file on your desktop to start the IDLE window. In the IDLE window, enter the following: 
      >>> from myro import *
  >>> upgrade("myro")
    If you get error messages about not having the proper permissions to write/upgrade files, you need to start python as an Administrator. The easiest way is to open an administrator command prompt (described above), navigate to your desktop, and run python "Start Python.pyw" in the command prompt window.)

    For the latest information on installing Myro, check the online website.
