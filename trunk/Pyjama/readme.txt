Pyjama directory

- Pyjama.sln: Solution Project for Pyjama, VS Express 2008
- src: Source code for Pyjama .NET/Mono code
- libs: needed for Pyjama Project

- python: IronPython compatible python modules
- po: Language translations for old version of GTK Pyjama
- modules: c# Libraries for Pyjama (Scheme and Conx)
- Highlighter: sample code for scanner/parser
- docs: documentation for misc code
